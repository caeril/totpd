totpd v0.2

Service Documentation


Everything is configured through environment variables. Here are the highlights:

TOTPD_DATA_PATH=/etc/example # /tmp is the default if this environment variable doesn't exist.
TOTPD_HTTP_PORT=31337        # default port is 8087

API Documentation

Add User Profile


POST /api/users

Request Body: { "organization": "Example, Inc.", "username": "bob@example.com" }

Response Body: { "message": "success", "uid": "58b2498b5592ee6d683d0347416d3548813fbcec0d145cc0cc753fab86dcde8d" }

Delete User Profile


DELETE /api/users/{uid}

Response Body: { "message": "success" }
Response Body: { "message": "failure" }

Validate User Code


GET /api/validate/{uid}?code={code}

Response Body: { "message": "valid" }
Response Body: { "message": "invalid" }
Response Body: { "message": "failure" }

Get User QR Code


GET /qrc/{uid}.png

Response Body: image/png of 512x512 pixel QR code
